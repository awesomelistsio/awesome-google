## Google APIs

- [Google Maps Platform](https://developers.google.com/maps) — APIs for maps, routes, and places.
- [Google Sheets API](https://developers.google.com/sheets/api) — Programmatically access and modify Google Sheets.
- [Google Drive API](https://developers.google.com/drive) — Integrate file storage and sharing functionality.
- [YouTube Data API](https://developers.google.com/youtube/) — Access YouTube video and channel data.
- [Google Calendar API](https://developers.google.com/calendar) — Access and manage calendar events.

## Google Cloud Platform (GCP)

- [Google Cloud Console](https://console.cloud.google.com/) — Web-based management for Google Cloud resources.
- [Google Kubernetes Engine](https://cloud.google.com/kubernetes-engine) — Managed Kubernetes service.
- [Google BigQuery](https://cloud.google.com/bigquery) — Data warehouse and analytics platform.
- [Google Cloud Storage](https://cloud.google.com/storage) — Scalable object storage.
- [Google Cloud Functions](https://cloud.google.com/functions) — Serverless compute functions.

## Google Marketing Tools

- [Google Analytics](https://analytics.google.com/) — Web analytics platform.
- [Google Ads](https://ads.google.com/) — Online advertising platform.
- [Google Search Console](https://search.google.com/search-console/) — Monitor website presence in Google search results.
- [Google Tag Manager](https://tagmanager.google.com/) — Manage website tags without editing code.
- [Google My Business](https://www.google.com/business/) — Manage business presence on Google search and maps.
